Game Rules

  • Each player starts with 5 dice.
  • Players take turns making bids about the total number of dice showing a specific face.
  • Bids must increase in quantity or face value.
  • If you think a bid is false, call "Liar!"
  • If the bid is true, the challenger loses a die.
  • If the bid is false, the bidder loses a die.
  • The last player with dice remaining wins the game.
